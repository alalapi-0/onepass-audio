# normalize_original.py

- 最后更新时间：2025-10-31 21:17:29 +0900

## 功能概述
提供原始稿件规范化的完整命令行工具，包括 Unicode 兼容字符映射、繁简转换、标点风格统一、前后附加内容清洗以及与 ASR 词级输出的对比分析。脚本支持生成丰富的统计数据和报告，帮助在对齐前发现潜在问题。

## 关键职责
- 定义多种管道阶段与统计数据类，串联文本加载、NFKC 归一、兼容字符映射、OpenCC 转换、标点替换和可疑字符检测。
- 支持通过 argparse 配置输入输出、风格和开关，并记录执行过程中的警告、替换次数与友好度指标。
- 输出 JSON/CSV 报告，提供字符频次、对齐友好度和差异分析结果，辅助人工审校。
